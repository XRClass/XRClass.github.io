About XR Class

Making EDUCATION more GLOBALLY ACESSIBLE

XR Class helps students, teachers, and school administrators obtain the benefits of having an in person class no matter how far away they may be.

increases excitement about education and can be tailored for pre-elementary to Collegiate and University classes.

This project allows for professors to have full control of their meetings like any other class management site. Additionally, they can view analytics on class attendance and performace and adjust their class meetings accordingly.

XR Class is a free software that allows for anybody with a Class URL to take any class in any school environment they wish. This project was developed using the following Softwares:

  • Unity
  • Photon
  • Agora SDK
  • Blender
  • PHPMailer
  • Clever-Cloud
  • Github Pages
  • Three js
  • Affectiva

Currently all Classes and Meetings held in the XR Class Application support 20 users concurrently. The goal of the next update is to create a dedicated server instead of using the free version of PhotonEngine.
